# Contributing to Electronics Hot Deals

Thank you for your interest in contributing! This document provides guidelines for contributing to the project.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ElectronicsHotDeals.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies as described in the README

## Code Style

### Python
- Follow PEP 8 guidelines
- Use type hints where appropriate
- Keep functions focused and well-documented

### JavaScript/React
- Use functional components with hooks
- Follow ESLint recommendations
- Keep components small and reusable

## Commit Messages

Use clear, descriptive commit messages:
- `feat: add new filter option for price range`
- `fix: resolve API timeout issue`
- `docs: update installation instructions`
- `style: improve mobile responsiveness`

## Pull Request Process

1. Update documentation if needed
2. Test your changes locally
3. Submit a PR with a clear description
4. Wait for review

## Questions?

Open an issue for any questions or suggestions!
